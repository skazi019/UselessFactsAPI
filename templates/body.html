Getting Started

Documentation

You can access the complete set of endpoints and their description in these formats: Swagger and Redoc

Here is a simple guide on how you can use these APIs

Instructions

Step 1: Call the /auth/registration/ endpoint to register your account and receive a Token. If you've already registered then call the /auth/login/ endpoint to receive the Token.

Step 2: While calling any of the v1 APIs, send the Token, received in the above steps, in the header as
Authorization: Token (the_token_received_without_paranthesies)

Step 3: To logout, call the /auth/logout/ endpoint without any body but send the Token in header as stated above.

Note: APIs are rate limited to 10/day/user
